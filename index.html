Дз по семантической структуре страницы

  • info header
  • info footer
  • info nav
  • info article
  • info aside
  • info section
  • info main

Вам нужно сделать основную структуру страницы используя семантические теги header, footer, nav, article, aside, section, main.

А так же abbr, address, blockquote, cite, q, strong strong, em, mark, dfn, ins, del для работы с текстом.

В блоке header нужно сделать навигацию при помощи nav.

В блоке footer нужно расположить address.

В блоке aside может быть один или несколько article .

В блоке main нужно сделать одну или несколько section.

В section уже нужно написать какой то текст и использовать все теги для работы с текстом.

Оцениваться будет правильность использования тегов.

Цитата: Если вы забыли значение какого нибудь из тегов, их можно найти здесь.

Для демонстрации тега blockquote напишу lorem:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Porro rem saepe, ipsum id molestias quos, enim impedit libero error laboriosam distinctio praesentium, deleniti quasi blanditiis perspiciatis repudiandae iusto dolorem assumenda?

Далее для демонстрации привожу следующие теги: курсив, маркированный тег для выделения текста, тег для терминов термин, подчеркнутый текст, зачеркнутый текст.

Рекомендуемая литература:

  • Универсальные атрибуты;
  • События;
  • Доступность;
  • Семантика;
  • SEO Oптимизация;
  • WAI-ARIA тандарт;
  • WAI-ARIA roles;
  • Open Graph.
Для обратной связи просьба писать на почту или LinkedIn.
