Дз по семантической структуре страницы

  • info header
  • info footer
  • info nav
  • info article
  • info aside
  • info section
  • info main

Вам нужно сделать основную структуру страницы используя семантические теги header, footer, nav, article, aside, section, main.

А так же abbr, address, blockquote, cite, q, strong strong, em, mark, dfn, ins, del для работы с текстом.

В блоке header нужно сделать навигацию при помощи nav.

В блоке footer нужно расположить address.

В блоке aside может быть один или несколько article.

В блоке main нужно сделать одну или несколько section.

В section уже нужно написать какой то текст и использовать все теги для работы с текстом

Оцениваться будет правильность использование тегов.

Если вы забыли значение какого нибудь из тегов, их можно найти здесь.
